### 코드 컨벤션!
1. 변수 camelCase ('_' )
    - (o) int playerNum = 0;
    - (x) int player_num = 0;
    - (o) string playerName = "asdf";
2. 함수, 클래스 등 나머지는 Pascal
    - (o) class MyClass
    - (x) class kebab-case
3. 작업은 씬을 분할하여 작업 후 마무리할 때 다같이 화면 공유하고 합치기
    - 김용준 작업 -> 오직 dev_YJKIM 씬에서만 (논의 없을시)
